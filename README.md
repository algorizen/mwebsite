# Maizuo

M-website project (SPA).

## Technology stack

- Gulp + Webpack
- Yo + Zepto + Better-scroll
- Handlebars
- Router (Custom router based on hash mode)

## Run

```
npm install
```

### Dev

```
npm run dev
```

### Prod

```
npm run build
```
